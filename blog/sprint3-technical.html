Control Flow and Loops

Here are some examples of statements and loops

If/Else Statement

 

if(ice-cream == 'delicious') {
    console.log('Ice cream is delicious')
}  else {
    console.log('Error, Ice cream is delicious')
}

Switch Statement

 

switch (ice-cream) {
    case 'chocolate':
        eatIceCream()
        break;
    case 'raisin':
        binIceCream()
        break;
}

For Loop

 

for (let i = 0; i < iceCream.length; i++) {
    iceCreamInTum += 1
}

Do While Loop

 

let i = 0
do {
    eatIceCream()
    i++
} while (i < iceCream.length)

To control the flow of the code is to direct it in a particular directions using different checks. By using an if/else statement or a switch statement, you can check to see if certain conditions are true, and execute different code blocks based on the result. In the switch statement above for example, we are checking the flavour of the ice cream and executing an action based on the result. Raisins should be illegal.

Loops are used to repeat a block of code until a condition becomes true. In the above Do While Loop, we continue to perform the eatIceCream() function until there is no ice cream left.
